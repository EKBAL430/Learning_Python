# Python Type Conversion

## 🧠 Core Idea

These functions convert data into different types.

Think:
Same thing → Different forms  
Like milk → curd → butter

---

# 📄 `List`

```markdown
- Ordered and changeable

### 🧠 Memory Trick:
👉 Shopping List — you can add/remove items anytime

### Example:
list("abc")  # ['a', 'b', 'c']
```

---

# 📄 `Tuple`

```markdown
A tuple is an ordered collection of items that **cannot be changed**.

## 🧠 Memory Trick

👉 Wedding Thali  
Once served, you cannot change items.

## Example
t = (1, 2, 3)
print(t)
```


---
# 📄 `set`

```markdown
A set is a collection of **unique elements**.

## 🧠 Memory Trick

👉 Entry gate  
Only one entry per person (no duplicates)

## Example
s = {1, 2, 2, 3}
print(s)  # {1, 2, 3}
```


---

# 📄 `dict`

```markdown
A dictionary stores data in **key-value pairs**.

## 🧠 Memory Trick

👉 Phonebook  
Name → Phone Number

## Example
d = {"name": "Aisha", "age": 20}
print(d["name"])
```


---

# 📄 `ord.md`

```markdown
ord() converts a character into its **ASCII/Unicode number**.

## 🧠 Memory Trick

👉 Every character has a secret ID

## Example
ord('A')  # 65
ord('a')  # 97
```
